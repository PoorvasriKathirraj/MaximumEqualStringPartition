MaximumEqualStringPartition

This project is a simple Java program that finds the maximum number of equal pieces that can be formed from a given string.
The program first accepts a string input from the user through the console.
It then counts the frequency of each character present in the string.
After calculating the frequencies, the program finds the Greatest Common Divisor (GCD) of those values.
The GCD represents the maximum number of identical substrings that can be created.
The program demonstrates basic concepts such as string handling, arrays, loops, and mathematical logic in Java.
This project was developed to practice problem-solving and improve understanding of Java programming concepts.
